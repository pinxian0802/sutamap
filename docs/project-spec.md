## 概念

日本地圖集點成就遊戲。玩家實際前往各地景點，以 GPS 打卡或照片上傳的方式收集地點，累積經驗值升等，集滿主題獲得稱號與徽章，並能與好友互相比較進度。

---

## 打卡機制

- **照片必填**：每次打卡必須上傳照片
- **GPS 驗證**：優先使用照片 EXIF GPS，若無則使用裝置即時 GPS
- **距離驗證**：依各主題設定不同允許半徑（見主題設定）
- **首次打卡才給 XP**：同一地點重複打卡只留紀錄，不累積 XP
- **照片處理**：前端抽取 EXIF GPS 後，清除全部 EXIF 資料再壓縮（最大 1MB），上傳至 Cloudflare R2

## XP 與等級系統

- XP 升等公式：`level = floor(0.5 + sqrt(0.25 + xp / 50))`
- 等級所需 XP：`xpForLevel(N) = N × (N-1) × 50`

| 等級 | 累計 XP |
|------|---------|
| Lv 1 | 0 |
| Lv 2 | 100 |
| Lv 3 | 300 |
| Lv 4 | 600 |
| Lv 5 | 1,000 |
| Lv 6 | 1,500 |

- 等級純顯示用，無額外解鎖效果
- 完成主題有額外 bonus XP（各 plan 實作時定義）

## 成就系統

- 集滿一個主題所有地點 → 自動頒發該主題的**稱號**與**徽章**
- 個人頁可選一個稱號作為「顯示稱號」，顯示在個人資料旁

---

## 地圖顯示

- 底圖：**CARTO Basemap**（Voyager 樣式）
- 密集區域自動**標記群集**（leaflet.markercluster），放大才展開
- 已打卡：顯示該主題顏色；未打卡：灰色
- **類別篩選**：左上角可切換顯示哪些主題的標記
- **好友模式**：切換後疊加所有好友的打卡記錄，不同好友用不同顏色的小圓點標示

## 權限分層

| 功能 | 未登入 | 已登入 | 已有好友 |
|------|--------|--------|---------|
| 瀏覽地圖、查看地點 | ✅ | ✅ | ✅ |
| 打卡、累積 XP | ❌ | ✅ | ✅ |
| 查看個人進度、徽章 | ❌ | ✅ | ✅ |
| 好友模式、排行榜 | ❌ | ❌* | ✅ |

> *排行榜全站榜不需好友即可查看，好友榜需要有好友

---

## 隱私設定

- **完全公開**：所有打卡記錄、進度、徽章對所有人可見
- 無任何隱私設定選項

---

## 社交系統

**加好友方式（四選一）：**
1. 搜尋用戶名
2. 輸入用戶 ID
3. 掃描 QR Code
4. 分享連結

**好友功能：**
- 查看好友的等級、已打卡地點、已獲徽章
- 在地圖上看到好友的打卡記錄（好友模式）
- 主題頁顯示「哪些好友也在集這個主題」

**排行榜（各有好友榜 + 全站榜）：**
1. XP 總量榜
2. 打卡次數榜
3. 徽章數量榜

## 頁面結構

| 路由 | 功能 | 需要登入 |
|------|------|---------|
| `/map` | 主地圖，類別篩選，好友模式 | 否（打卡需要） |
| `/checkin/[id]` | 打卡流程 | 是 |
| `/profile` | 個人頁：XP、等級、稱號選擇、徽章、進度 | 是 |
| `/profile/[userId]` | 他人公開頁面 | 否 |
| `/categories` | 主題列表，各主題進度，好友也在集的主題 | 否 |
| `/friends` | 好友列表，加好友，接受/拒絕申請 | 是 |
| `/leaderboard` | 排行榜（XP/打卡/徽章 × 好友/全站） | 否（好友榜需登入） |
| `/auth/login` | 登入 | 否 |
| `/auth/register` | 註冊 | 否 |

## 技術架構

| 層 | 技術 |
|---|---|
| 框架 | Next.js 15（App Router, TypeScript） |
| 地圖 | Leaflet.js + CARTO Basemap + leaflet.markercluster |
| Auth + Database | Supabase（PostgreSQL + RLS） |
| 圖片儲存 | Cloudflare R2 |
| UI | Shadcn/ui + Tailwind CSS（Mobile-first） |
| PWA | next-pwa（可安裝至手機桌面） |
| EXIF 解析 | exifr（純前端，抽 GPS 後清除 EXIF） |
| 照片壓縮 | browser-image-compression（前端壓縮至 1MB） |
| QR Code | qrcode.react |
| 部署 | 待確認（Vercel / Netlify / Cloudflare Pages） |

## 資料庫 Schema

```
categories         主題類別（名稱、顏色、圖示、XP、打卡半徑）
locations          地點（所屬主題、名稱、座標、都道府縣）
checkins           打卡記錄（用戶、地點、照片URL、GPS座標、距離、是否首次）
user_profiles      用戶資料（用戶名、總XP、等級、顯示稱號）
titles             稱號定義（名稱、說明、所屬主題）
badges             徽章定義（名稱、說明、圖示、所屬主題）
user_titles        用戶已獲稱號
user_badges        用戶已獲徽章
friendships        好友關係（申請者、被申請者、狀態 pending/accepted/rejected）
```
